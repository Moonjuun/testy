# Google AdSense 광고 단위 생성 가이드

## 📋 개요

현재 프로젝트에서 사용 중인 광고 크기별로 **별도의 광고 단위**를 생성해야 합니다.
AdSense는 하나의 슬롯이 하나의 크기만 지원하므로, 각 크기마다 새로운 광고 단위를 만들어야 합니다.

## 🎯 필요한 광고 단위 (총 5개)

### 1. 336x280 (큰 사각형)

- **사용 위치**: MBTI 컬렉션과 에디터 픽 사이, 테스트 페이지, 타로 페이지 등
- **타입**: 디스플레이 광고
- **크기**: 고정 크기 → 336x280 선택

### 2. 300x250 (중간 사각형)

- **사용 위치**: 모바일에서 MBTI 컬렉션과 에디터 픽 사이, 테스트 페이지, 타로 페이지 등
- **타입**: 디스플레이 광고
- **크기**: 고정 크기 → 300x250 선택

### 3. 320x50 (모바일 배너)

- **사용 위치**: 모바일 하단 고정 배너
- **타입**: 디스플레이 광고
- **크기**: 고정 크기 → 320x50 선택

### 4. 300x600 (사이드 광고 - 왼쪽)

- **사용 위치**: 데스크탑 왼쪽 사이드바 (2xl 이상)
- **타입**: 디스플레이 광고
- **크기**: 고정 크기 → 300x600 선택

### 5. 300x600 (사이드 광고 - 오른쪽)

- **사용 위치**: 데스크탑 오른쪽 사이드바 (2xl 이상)
- **타입**: 디스플레이 광고
- **크기**: 고정 크기 → 300x600 선택

## 📝 단계별 생성 방법

### Step 1: AdSense 대시보드 접속

1. [Google AdSense](https://www.google.com/adsense) 접속
2. 로그인 후 대시보드로 이동

### Step 2: 광고 단위 생성 시작

1. 왼쪽 메뉴에서 **"광고"** 클릭
2. **"광고 단위"** 클릭
3. **"새 광고 단위 만들기"** 버튼 클릭

### Step 3: 광고 단위 형식 선택

⚠️ **중요**: 다음 화면에서 **"디스플레이 광고"**를 선택하세요!

- ✅ **디스플레이 광고**: 고정 크기의 배너 광고 (우리가 사용하는 타입)
- ❌ 인피드: 콘텐츠 피드에 자연스럽게 삽입 (사용 안 함)
- ❌ 인아티클: 기사 중간에 삽입 (사용 안 함)
- ❌ 멀티플렉스: 여러 광고를 한 번에 표시 (사용 안 함)

### Step 4: 광고 단위 설정

1. **광고 단위 이름** 입력
   - 예: `테스티_336x280`, `테스티_300x250`, `테스티_320x50`, `테스티_사이드_왼쪽`, `테스티_사이드_오른쪽`
2. **광고 크기** 선택

   - **"고정 크기"** 선택 (반응형 아님)
   - 아래 크기 중 하나 선택:
     - 336x280 (큰 사각형)
     - 300x250 (중간 사각형)
     - 320x50 (모바일 배너)
     - 300x600 (사이드바)

3. **"만들기"** 버튼 클릭

### Step 5: 슬롯 ID 복사

1. 생성된 광고 단위 페이지에서 **"슬롯 ID"** 복사
2. `constants/ads.ts` 파일의 해당 TODO 주석 부분에 슬롯 ID 입력

## 📋 생성 체크리스트

각 광고 단위 생성 후 아래 항목을 확인하세요:

- [ ] 336x280 (큰 사각형) - `DESKTOP_336x280`
- [ ] 300x250 (중간 사각형) - `MOBILE_300x250`
- [ ] 320x50 (모바일 배너) - `MOBILE_320x50`
- [ ] 300x600 (사이드 왼쪽) - `SIDE_LEFT_300x600`
- [ ] 300x600 (사이드 오른쪽) - `SIDE_RIGHT_300x600` (이미 "2714274112" 있지만 확인 필요)

## ⚠️ 주의사항

1. **반드시 "디스플레이 광고" 타입 선택**: 다른 타입(인피드, 인아티클 등)은 사용하지 않습니다.
2. **고정 크기 선택**: 반응형이 아닌 고정 크기를 선택해야 합니다.
3. **각 크기마다 별도 생성**: 같은 크기라도 위치가 다르면(예: 사이드 왼쪽/오른쪽) 별도로 생성해야 합니다.
4. **슬롯 ID는 정확히 입력**: 슬롯 ID를 잘못 입력하면 400 에러가 발생합니다.

## 🔍 확인 방법

광고 단위 생성 후:

1. `constants/ads.ts` 파일에 슬롯 ID 입력
2. 프로덕션 환경에서 테스트
3. 브라우저 개발자 도구에서 400 에러가 사라졌는지 확인
4. 광고가 정상적으로 표시되는지 확인
